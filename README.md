# Full-Stack Weather App 🌤️

A lightweight, responsive weather application built with a Python/Flask backend and a Vanilla JavaScript frontend. It fetches real-time weather data from the OpenWeatherMap API and displays it in a clean, modern UI.

## ✨ Features
* **Real-time Weather Data:** Accurate temperature, humidity, wind speed, and "feels like" metrics.
* **Secure Backend:** API calls to OpenWeatherMap are handled securely through a Flask server, keeping API keys hidden from the client.
* **Modern UI:** Responsive design with dynamic weather icons and smooth CSS animations.
* **Keyboard Support:** Trigger searches instantly using the Enter key.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Python, Flask
* **API:** OpenWeatherMap API
* **Environment Management:** `python-dotenv`
